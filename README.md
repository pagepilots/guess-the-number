# Guess the Number

i made this when i was learning loops. it's not impressive but it works.

## how it works
computer picks a number between 1 and 100
you guess
it tells you too high or too low
you keep guessing until you get it

## features
- counts how many guesses you took
- yells at you if you type letters
- gives up after 10 tries (optional mode)

## run it
python guess.py

## why i made this
teacher said make something with random module. this is that thing.
